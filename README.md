# Kaspi Товар-Аналитик

Небольшой сайт для отбора товаров перед закупом из Китая и продажей на Kaspi.

## Что умеет

- принимает ссылку на товар Kaspi;
- пытается понять название товара из ссылки;
- считает карго по ставке 3.5 $/кг;
- считает себестоимость, прибыль, маржу и рекомендуемую цену;
- дает решение: брать в тест, тестировать осторожно или пока не брать;
- готовит короткий текст для карточки товара;
- дает чек-лист перед закупом.

## Как пользоваться

1. Вставьте ссылку Kaspi.
2. Заполните цену на Kaspi, минимальную цену конкурента, закупочную цену и вес.
3. Добавьте количество конкурентов, отзывы, рейтинг и продажи поставщика.
4. Смотрите блок “Решение” и расчет маржи.
5. Кнопкой “Скопировать анализ” заберите текст для заметок или таблицы.

Первый вариант работает полуавтоматически, потому что Kaspi, 1688 и Taobao могут ограничивать автоматический сбор данных. Это безопаснее для магазина и быстрее для запуска.
